S.N: 02
The basics of Chrome Extension

The basics about chrome extensions.
Here the basic extension prints the predefined message in console and changes the background color of 'p' tag and replace the src of img tag to any other src.->content.js.
The manifest.json file is like the configuration file containing the info of the extension.

How to add chrome extensions in local machine
  • visit chrome://extensions/
  • Turn on the developer mode
  • Click on the load unpacked
  • Select the folder where the all necessary file exists.

S.N: 04
BlueRay Filter Extension

This chrome extension enables the blueray filter on the basis of time.

Steps:
  • Create the working directory. Here,chrome-ext/blueray
  • Create all script folder and add two new files
    • content_script.js
    • background_script.js
  • Add a new file and rename to manifest.json
    • { "manifest_version" : 2, "name": "BlueRay Filter", "version": "1.0", "content_scripts":[ { "matches": [ "" ], "js": ["scripts/content_scripts.js"] } ], "background": { "scripts": ["scripts/background_scripts.js"] }, "browser_action":{ "default_icon": "img.png" }, "permissions": [ "activeTab" ] }
  • content_scripts.js
  • background_scripts.js
